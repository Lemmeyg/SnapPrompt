SnapPrompt User Guide

Version 1.4.0

SnapPrompt is a text snippet management extension designed to streamline your workflow by allowing you to quickly save, insert and organize frequently used text into any text field across the web. The extension is ideal for managing common replies, email signatures, AI prompts, code snippets, and other repetitive text inputs.

Key Features

Clipboard Copy Functionality
Each saved snippet includes a dedicated copy button that transfers the text directly to your clipboard without inserting it into a text field. This feature provides universal compatibility with applications outside the browser, including desktop software, email clients, and document editors.
Keyboard Shortcut Integration
Configure keyboard shortcuts (Ctrl+Shift+1 through Ctrl+Shift+4) to instantly insert your most frequently used snippets within Chrome. Shortcuts are assigned to the first four snippets in alphabetical order and can be customized through Chrome's extension shortcuts interface at chrome://extensions/shortcuts. For use in other applications, use the copy button to transfer snippets to your clipboard.
Context Menu Text Capture
Select text on any webpage, access the browser context menu, and choose the "Save in SnapPrompt" option to create a new snippet instantly. This feature eliminates the need to manually copy and paste text into the extension interface.
Productivity Analytics
The extension automatically tracks the number of keystrokes saved through snippet usage and converts this data into estimated time savings. Achievement milestones are displayed at specific intervals to provide insight into your productivity improvements.
Theme Customization
Toggle between professional dark and light interface themes through the settings menu. Both themes feature rounded corners, smooth animations, and carefully selected color palettes designed for extended use and visual comfort.

Getting Started

Creating Snippets
Click the SnapPrompt extension icon in your browser toolbar to open the management interface. Enter a descriptive label and the desired text content, then click the "Save Prompt" button. The extension supports up to ten saved snippets at any given time.
Using Snippets
Snippets can be inserted into text fields through multiple methods: selecting from the browser context menu, using configured keyboard shortcuts (Ctrl+Shift+1 through Ctrl+Shift+4), or clicking the copy button for clipboard-based insertion into any application.
Managing Your Collection
Hover over any snippet to reveal the edit and delete action buttons. Snippets can be reordered through drag-and-drop functionality to prioritize frequently used items. Click on any snippet to view its complete text content in an expanded overlay view.
Data Portability
Access the Import and Export functions through the settings menu to create backup copies of your snippet collection. Exported data is saved in JSON format and can be imported on other devices or browsers to synchronize your snippet library.

Best Practices

  • Utilize the clipboard copy functionality when working with applications outside the browser environment where direct text insertion is not available.
  • Consider switching between light and dark themes based on your working environment and time of day to reduce eye strain during extended usage.
  • Create regular backups of your snippet collection using the export functionality to prevent data loss during browser updates or system changes.
